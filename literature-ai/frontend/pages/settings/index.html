📖 使用说明

这套软件能做什么

  • 本地文献库管理
  • 在线检索文献（arXiv / OpenAlex / PubMed / Semantic Scholar）
  • DOI 下载并自动入库
  • PDF 解析（Docling + GROBID）
  • DFT / 电化学 / 机理 / 写作卡片抽取
  • AI 分类（计算型/实验型/综述型）
  • 图片提取 + VLM 分类 + 数值提取
  • 分类感知 RAG 检索 + 写作
  • 外部 AI 审稿工作台
  • MCP 对接 IDE 内 AI（Cursor / VS Code / Claude 等）

启动方式

在项目目录运行：

docker compose up -d --build

或者本地开发模式：

cd backend python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

常用网页入口

  • 健康检查：/api/health
  • 文献库：/pages/literature_library/index.html
  • 论文详情：/pages/paper_detail/index.html?id=<paper_id>
  • AI Writer：/pages/ai_writer/index.html
  • 外部审稿：/pages/literature_library/index.html?tab=review
  • 设置：/pages/settings/index.html

如果网页打不开

  1. 先确认服务是否在运行：docker ps 或检查本地进程
  2. 访问 /api/health，如果健康检查失败，问题在后端
  3. 查看后端日志：docker compose logs backend --tail=200
  4. 局域网访问需确认：防火墙放行 8000 端口 + 同一网段

IDE AI 连接

  1. 在本页面「API 配置」中配置好 MCP Keys
  2. 切换到「IDE 连接」标签页，复制一键提示词
  3. 把提示词发给你的 AI（Cursor / Claude / Gemini 等）
  4. AI 会自动配置 MCP 连接，之后可以直接操作你的文献库

推荐工作流

  1. 本地系统负责存储与解析
  2. IDE 里的 AI 通过 MCP 读写文献与审稿候选项
  3. 外部 AI 通过审稿工作台参与复核
  4. 最终由你或审核 AI 决定哪些修改进入正式数据
